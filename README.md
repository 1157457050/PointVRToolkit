# PointVRToolkit
虚幻引擎4虚拟现实项目工具包

* 纯蓝图项目，基于UE4.23

* 包含基本交互功能：抓取、按钮、滑块、旋转曲柄、旋钮、开关门、弓箭（双手交互）、固定传送点、激光抓取、3D菜单控制

* 2020-3-25 项目停止更新（个人已退出VR行业）
